# Movie Recommendation System 🎬

This project is a Movie Recommendation System developed using Python and Machine Learning concepts.

The system recommends movies similar to the movie selected by the user using Content-Based Filtering.

## Technologies Used

* Python
* Pandas
* Scikit-learn
* CountVectorizer
* Cosine Similarity

## Features

* User movie input
* Top 5 movie recommendations
* Similarity-based recommendation system

## Machine Learning Concept

* Content-Based Filtering

## Example

Input:
Avengers

Output:

* Iron Man
* Thor
* Captain America
* Free Guy
* PK
